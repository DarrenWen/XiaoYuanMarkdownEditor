# 小猿MarkDown编辑器
---
> 是一款功能简单的MarkDown文件编辑器，可以实现Markdown文件的编辑和保存，该编辑器为教学案例，完全开源，大家可以根据自己的实际情况下载使用！
> 
> [教程地址](https://blog.csdn.net/fwj380891124/article/details/119214040)
> 1. 下载代码
> 2. 按教程全局安装好electron
> 1. 执行```npm start```查看效果
